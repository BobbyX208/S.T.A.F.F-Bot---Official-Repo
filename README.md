## Features
- 🔄 Auto-updates from GitHub
- 🎫 Ticket system with categories
- ✅ User verification (button/captcha)
- 🛒 Catalog/shop with tickets
- 🛡️ Moderation commands
- 📝 Event logging
- 🎭 Role menus

## Quick Start

```bash
# Clone
git clone https://github.com/yourusername/staff-bot.git
cd staff-bot

# Setup
python setup.py

# Edit config
cp .env.example .env
nano .env  # Add your token

# Run
python main.py
```

Commands

· /help - Show help menu
· /ping - Bot latency
· /uptime - Bot uptime

Admin commands in each cog.

Requirements

· Python 3.10+
· SQLite3
· Discord bot token
